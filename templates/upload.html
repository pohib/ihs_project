Справочник полей

name

Название правила. Если внутри есть запятые, используйте "кавычки".

source_ip / dest_ip

IP или подсети. Формат: 192.168.1.1 или 10.0.0.0/24.

port

Число (1-65535). Оставьте пустым для протоколов ICMP/ANY.

action

Только два значения: ALLOW или DENY.

Почему файл может не загрузиться?

  • • Разделитель в CSV — запятая. Если ваш Excel ставит точку с запятой (;), файл не прочитается.
  • • В названиях (name) не должно быть лишних кавычек, если вы не экранируете всё поле целиком.
  • • Протоколы пишутся капсом: TCP, UDP, ICMP, ANY.
